# PHP 實作：留言板

以 PHP 開發出 API，接著用 JavaScript 寫的前端可以串接的留言板練習。

[Demo](http://rkcfkrrt.tw/board/)

![使用截圖](https://images.plurk.com/1SdLp3yGUNk8pdmoBrWLPw.png)

## 留言板功能

- 留言板的 API，可新增留言以及顯示留言
- 前端頁面串接自己寫的 API
- 「載入更多」的功能

## 使用技術

- JavaScript、jQuery
- PHP
- HTML
- CSS、Bootstrap
- 支援 RWD

## 專案結構

- api_add_comments.php
- api_comments.php
- index.html
- index.js
